Etiquetas

La libreria Render permite que no se mezcle PHP y HTML, haciendo uso de etiquetas, un ejemplo de esto es muestra a continuación: 
La etiqueta: {{open_tags}}variable{{close_tags}}, se transforma en: {{variable}}

Limpieza de etiquetas

Las etiquetas sin un valor asignado (como esta: 
{{no_asignado}}
) son automáticamente borrados a menos que se indique lo contrario, mediante comando: 
$render->setClear(false);

Bucles

Para mostrar el contenido de un array, es necesario indicar el inicio y el cierre del bucle con la etiqueta: 
  <!--{{#llave_asociada}}-->
Y todo lo que se encuentre dentro de esas etiquetas, será repetido, haciendo los respectivos reemplazos.
En caso de que se este usando un array sin especificar sus claves (como por ejemplo: 
array('C1', 'C2', 'C3')
), se podrá acceder a los elementos este, mediante el comodín 
{{open_tags}}variable{{number}}
El libro: {{name}} tiene {{pages}} páginas
Tiene los capítulos:
  • {{number}}
En el ejemplo anterior se observa que el segundo libro no tiene disponible sus capítulos, por lo que el apartado "Capítulos" esta en blanco. Render no permite realizar condiciones, ya que se considera que para solucionar este tipo de problemas, existen otras formas como el uso de javascript ;).
